## 📊 System Architecture

### 4-Layer Hierarchical Mesh

```
Layer 1 (Queen): Aggregate Swarm State
├── HexagonLayer visualization
├── Coordination of 8-15 specialist agents
└── Real-time decision aggregation

Layer 2 (Specialists): Agent ROAM Metrics
├── ScatterplotLayer (3D points)
├── 8 specialist agents with confidence scores
└── Risk/Opportunity/Action/Mitigation tracking

Layer 3 (Memory): Vector Search Connections
├── ArcLayer (256+ connections)
├── HNSW-indexed search (150x-12,500x faster)
└── Hybrid memory backend (ChromaDB + AgentDB)

Layer 4 (Execution): Real-time WebGL Streaming
├── StreamLayer updates
├── WebSocket connection (port 3001)
└── MCP/MPP integration
```

## 🔄 Method Pattern Protocol Factors

### Iterate/Flow/PI/Spike/Sprint/Sync

| Method | Duration | Purpose | WSJF Weight | Coverage Impact |
|--------|----------|---------|-------------|-----------------|
| **Iterate** | 30-60 min | Continuous improvement cycle | 1.0x | +5-10% |
| **Flow** | 2-4 hours | Uninterrupted deep work | 1.5x | +15-25% |
| **PI** (Program Increment) | 2 weeks | Major feature development | 2.0x | +30-40% |
| **Spike** | 1-2 days | Research/exploration | 0.5x | Variable |
| **Sprint** | 1 week | Focused delivery | 1.8x | +20-30% |
| **Sync** | 15-30 min | Alignment meeting | 0.2x | +2-5% |

## 🎯 WSJF Prioritization Engine

### Formula

```
WSJF = (Business Value + Time Criticality + Risk Reduction) / Job Size

Where:
- Business Value: 1-10 (inverse of success rate)
- Time Criticality: 1-10 (based on episode count)
- Risk Reduction: 1-10 (ROAM-based)
- Job Size: 5-20 minutes (DoR time budget)
```

### Current Priorities (from ay wsjf)

```
Priority Queue (WSJF Score):
  3.00 - orchestrator (standup)  - Deploy Deck.gl visualizations ⭐ HIGH
  1.50 - innovator (retro)       - Test infrastructure connectivity
  1.00 - seeker (replenish)      - Complete environment configuration
  1.00 - assessor (wsjf)         - AISP integration
  0.75 - intuitive (synthesis)   - Codebase refactoring
  0.75 - analyst (refine)        - Documentation updates
```

### Dynamic Reprioritization

Every iteration cycle automatically recalculates WSJF based on:
1. Success rate changes
2. New episodes in AgentDB
3. ROAM risk updates
4. Layer health metrics
