## 2. iOS / Android Prod Maturity

### Current state
- `apps/summerjobswap/` is a Capacitor v1.0 shell.
- `src/index.html` redirects to `https://summerjobswap.com`.
- `android/app/build.gradle` has no release signing config.
- No `google-services.json` → Firebase push not applied.
- No Detox config, no fastlane, no native E2E.
- `versionCode: 1`, `versionName: "1.0"` unchanged.

### Verdict
Web-redirect store presence only. The native binary is **not shippable** as a product.

### Why this is acceptable now
- Mobile strategy is currently "store listing + web redirect".
- Playwright web gates (`app-store-redirection.spec.ts`, `epic-40-app-store-gate.spec.ts`) validate the funnels.
- Moving to native features requires scoped initiative: signing, Firebase/APNs, in-app UX, CI macOS runners.

### Recommended ROAM risk
```yaml
id: R-MOBILE-01
roam: Accepted
severity: medium
owner_layer: mobile
description: >
  iOS/Android Capacitor shells redirect to web. No release signing, Firebase,
  Detox, or fastlane. Native binary is not shippable.
```
