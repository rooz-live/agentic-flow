## Defined-Risk Options Strategies

### SOXL

**Bull Put Spread**:
- Structure: Sell OTM put + Buy further OTM put
- Max Risk: Max loss = spread width - credit received
- Max Reward: Max gain = credit received
- Use Case: Neutral to bullish, collect premium

**Bear Call Spread**:
- Structure: Sell OTM call + Buy further OTM call
- Max Risk: Max loss = spread width - credit received
- Max Reward: Max gain = credit received
- Use Case: Neutral to bearish, collect premium

**Iron Condor**:
- Structure: Bull put spread + Bear call spread
- Max Risk: Max loss = spread width - net credit
- Max Reward: Max gain = net credit received
- Use Case: Low volatility, range-bound

**Calendar Spread**:
- Structure: Sell near-term option + Buy longer-term option (same strike)
- Max Risk: Max loss = net debit paid
- Max Reward: Max gain = varies with time decay
- Use Case: Earnings volatility crush

**Note**: Use options profit calculator: https://www.optionsprofitcalculator.com/
