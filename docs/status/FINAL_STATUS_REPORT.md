- **Async Generation**: `await generate(messages)`
- **Streaming**: `async *generateStream(messages)` generator pattern

- **Chat Template**: GLM-4 format with [System], [User], [Assistant] sections
- **Model Management**: Initialize, load, unload, capability checks

**Usage Example**:
```typescript

  modelId: 'glm-4.7-reap-50',
  temperature: 0.7,
  maxTokens: 1024
});

const response = await llm.generate([
  { role: 'system', content: 'You are a helpful AI assistant.' },
  { role: 'user', content: 'Write a Python function to calculate Fibonacci.' }
]);

console.log(response.content);
await llm.unload();
```
