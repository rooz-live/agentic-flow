### Bridging Shell → Rust
Current shell-based validators (`validation-core.sh`, `validation-runner.sh`) can now be incrementally migrated to use the Rust domain:

```rust
use rust_core::validation::{EmailDocument, EmailValidatorService};

let service = EmailValidatorService::new();
let report = service.validate(&doc)?;
println!("Verdict: {} (%/# = {})", report.overall_verdict, report.coverage_snapshot());
```
